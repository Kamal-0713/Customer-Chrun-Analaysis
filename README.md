# 📊 Customer Churn Analysis Dashboard (Power BI)

## 📌 Project Overview

Customer churn is one of the biggest challenges for businesses. This project uses **Microsoft Power BI** to analyze customer data and identify the factors that contribute to customer churn. The interactive dashboard provides valuable insights that help businesses improve customer retention and make data-driven decisions.

---

## 🎯 Objectives

- Analyze customer churn patterns.
- Identify key factors affecting customer retention.
- Visualize customer demographics and behavior.
- Track churn rate using interactive dashboards.
- Support business decisions through data insights.

---

## 🛠️ Tools & Technologies

- **Microsoft Power BI**
- **Power Query**
- **DAX (Data Analysis Expressions)**

- 📌 Customer Tenure Analysis
- 📌 Interactive Filters & Slicers

---

## 📊 Key Insights

- Customers with **month-to-month contracts** have a higher churn rate.
- Customers with **higher monthly charges** are more likely to churn.
- Long-term customers generally show better retention.
- Contract type and payment methods significantly influence churn behavior.
- Interactive visualizations make it easy to explore customer segments.
